# Websiteleri

Bu repository birden fazla web sitesi projesini içerir.

## Projeler

- **web-sitesi** - Next.js ile geliştirilmiş web sitesi
- **mafsoft-web-sitesi** - Mafsoft için geliştirilmiş Next.js web sitesi

## Yeni Proje Ekleme

Yeni bir web sitesi eklemek için:
1. Yeni bir klasör oluşturun
2. Proje dosyalarınızı bu klasöre ekleyin
3. Commit ve push yapın
